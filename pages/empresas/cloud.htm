¿Qué es Cloud?

Cloud es un servicio que permite el almacenamiento y el acceso a sistemas informáticos y aplicaciones a través de internet. Así, las empresas y los particulares con un sitio web ya no necesitan disponer de un servidor físico para almacenar y acceder a sus datos.

Los usuarios del cloud pueden consultar los archivos, programas y datos desde cualquier dispositivo, por lo que no es necesario almacenarlos en el disco duro de un ordenador o en un servidor dedicado.

Accesibilidad

Descarga la aplicación en tu smartphone o tablet para tener siempre tus datos a mano desde cualquier lugar y a cualquier hora. Podrás acceder a tus datos en el cloud, desde cualquier navegador en cualquier dispositivo.


Sincronización

Cloud te acompaña en tu jornada laboral. Elige qué archivos o carpetas deseas sincronizar con el cloud mediante un software para tu PC Windows o Mac. Cualquier cambio que hagas online o local se sincroniza automáticamente.

Colaboración

Crea y administra cuentas individuales protegidas con contraseña. Comparte proyectos con solo un link y otorga a las personas derechos de solo lectura edición. Establece una fecha de vencimiento a los links compartidos, y limita el número de descargas.

Almacenamiento

Organiza y accede a los archivos adjuntos del correo electrónico fácilmente. Transfiérelos cómodamente desde cualquier cuenta a tu almacenamiento en Cloud. Simplemente reenvía el correo electrónico a Cloud y este guardará todos los archivos.

Digitalización

No dependas de los documentos en papel. Escanealos con la aplicación Cloud y guárdalos en formato PDF con reconocimiento de texto. Es una forma sencilla de digitalizar cualquier cosa, desde las facturas hasta los contratos, para que sean más fáciles de encontrar.

Protocolos

Integra Cloud en tus sistemas utilizando SFTP/FTPS, WebDAV, SMB/CIFS, rsync, SCP o Git. También puedes realizar una copia de seguridad automática de los datos de tu memoria conectada a la red (NAS). Cloud es compatible con una serie de discos duros de red.

Los diferentes tipos de Cloud

Existen cuatro tipos principales de servicios cloud: el cloud privado, el cloud público, el cloud híbrido y el multicloud.

Cloud Privado

El cloud privado o «private cloud» consiste en un servidor dedicado accesible desde cualquier dispositivo y que gestiona, por ejemplo, los datos informáticos para los empleados de una misma empresa. El cloud privado puede alojarse en las propias instalaciones o con un proveedor tercero en un datacenter.

El cloud dedicado ofrece ventajas como la flexibilidad y la adaptación a las necesidades de los usuarios, especialmente en materia de uso y rendimiento. De este modo, usted tendrá un mayor control sobre los datos y la gestión del servidor, dispondrá de redes privadas y podrá acceder a recursos dedicados. Este tipo de alojamiento también le ofrece un alto nivel de confidencialidad y sistemas de seguridad optimizados.

Cloud Público

El cloud público o «public cloud» es una opción muy fácil de utilizar para las empresas, ya que el proveedor de servicios cloud es el que se encarga del alojamiento en su propia infraestructura. En la modalidad de cloud público, OVHcloud se encarga de las actualizaciones de los sistemas, la gestión del servidor y la seguridad de los datos.

El cloud público ofrece múltiples ventajas desde un punto de vista económico, ya que usted solo pagará por las funcionalidades que utilice, cuando las utilice. Además, se trata de una opción flexible y puede estructurarse de diferentes maneras.

Cloud Híbrido

El cloud híbrido o «hybrid cloud» es una opción ágil que combina las ventajas de los servicios de cloud privado y cloud público. Este tipo de infraestructura incluye intercambios de información entre el cloud privado y el cloud público con una conexión encriptada.

Al transferir las cargas informáticas entre los diferentes tipos de cloud (privado y público) en función de las necesidades y los costes, favorece una mayor flexibilidad.

Multicloud

El servicio multicloud incluye varios servicios cloud del mismo tipo, tanto privados como públicos, a diferencia de lo que sucede en el caso del cloud híbrido, que sí permite combinar dos tipos de servicios cloud diferentes.
